# Siyuan Utils - Local Scripts for Siyuan

已实现的功能（部分源码未更新）：
- 题头图位置调整
- 浮动工具条
- @呼出块属性菜单
- 快捷搜索框（暂时可支持豆瓣书籍信息搜索和导入）
- 内容块批注评论
