# 思源笔记图标 Jewel

在思源笔记官方icon主题 material 的基础上重绘了文件树、段落、对齐等 icon，主要是统一下图标风格。然后恬不知耻的取名叫做 **Jewel（珠宝）**

使用方法：下载文件夹，复制至软件 icon 目录下，然后在 设置-外观-图标 中启用
